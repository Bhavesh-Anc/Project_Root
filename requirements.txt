#
# 2. Activate virtual environment:

#    python -m pip install --upgrade pip
#
# 4. Install requirements:
#    pip install -r requirements.txt

# ==================== TROUBLESHOOTING ====================

# Common Installation Issues and Solutions:

# Issue 1: Microsoft Visual C++ errors on Windows
# Solution: Install Microsoft C++ Build Tools
# https://visualstudio.microsoft.com/visual-cpp-build-tools/

# Issue 2: Compilation errors with advanced ML libraries
# Solution: Use conda instead of pip for problematic packages
# conda install -c conda-forge xgboost lightgbm catboost

# Issue 3: SSL certificate errors
# Solution: Upgrade certificates or use --trusted-host
# pip install --trusted-host pypi.org --trusted-host pypi.python.org

# Issue 4: Permission errors
# Solution: Use --user flag
# pip install --user -r requirements.txt

# Issue 5: Memory errors during installation
# Solution: Install packages one by one or increase virtual memory
